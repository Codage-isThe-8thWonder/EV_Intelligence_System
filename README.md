# EV Pulse 🚗⚡

FastAPI • PostgreSQL • Machine Learning • Streamlit • Docker • CI/CD

An Electric Vehicle Intelligence & Predictive Analytics Platform designed to monitor EV health, analyze performance, predict failures, and generate intelligent maintenance recommendations.

---

## 📖 Project Overview

EV Pulse is an end-to-end EV monitoring and predictive maintenance platform built using FastAPI, PostgreSQL, Machine Learning, and Streamlit.

The platform allows users to register vehicles, upload EV telemetry datasets, visualize vehicle performance through interactive dashboards, and generate predictive insights such as Remaining Useful Life (RUL), Failure Probability, and Component Health Score.

By combining data analytics with machine learning, EV Pulse helps identify potential issues before failure and provides actionable recommendations for improving vehicle reliability and battery health.
